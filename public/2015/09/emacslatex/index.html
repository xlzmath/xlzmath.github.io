Use emacs to edit latex code

What is emacs?

Emacs is a elisp interpreter equipped with some pre-compiled text edit features. This means that it is a text editor which you can hack in any direction you want. But we are just someone on the street, we know nothing about lisp programming. Yet no problem, we don’t have to, it is already done by the community, they call these hacks packages.

Pros and cons (why should I even consider to switch from my favorite IDE to this software even older than me?)

  • Pros
    • emacs adapts you (by its definition)
    • very efficient for repetitive text processing
    • fully customizable
    • somebody even claims that emacs change their life
  • Cons
    • painful learning curve
    • strange short cuts (especially you have to forget about Ctrl+c, Ctrl+v)
    • not very cross-platform since it is designed for *nux system, os-x is still ok, lesser windows support (there are walkarounds)
